What is Console ?

Console is an object which provides access to the browser debugging console . It is not used for user only for Coder.

What is of Console?

The console is an operating system window where users interact with the operating system or with a text-based console application by entering text input through the computer keyboard, and by reading text output from the computer terminal.

How to open Console?

By using Ctrl + Shift + I we open console or by clicking right button on mouse and then click to inspect console is open
